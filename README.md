## 📖 Overview

**SmartSchool LMS** is a complete, AI-powered Learning Management System designed specifically for Rwandan schools. It provides three distinct dashboards for **Admins**, **Teachers**, and **Students** to manage quizzes, track progress, and improve educational outcomes.

> 🇷🇼 **Built for Rwanda Education System** - REB Curriculum Compliant

---

## ✨ Features

### 👑 Admin Dashboard
- ✅ **User Management** - Add, edit, or delete teachers and students
- ✅ **Quiz Oversight** - View all quizzes created across the school
- ✅ **Analytics** - Track overall performance, top performers, and struggling students
- ✅ **Password Reset** - Send password reset emails to users

### 👨‍🏫 Teacher Dashboard
- ✅ **Create Quizzes** - Add unlimited questions with 4 options each
- ✅ **Question Bank** - Manage and organize questions
- ✅ **Student Performance** - View individual student scores and class averages
- ✅ **Timer Settings** - Set time limits for each quiz

### 👨‍🎓 Student Dashboard
- ✅ **Take Quizzes** - Interactive quiz-taking experience
- ✅ **Instant Feedback** - See correct/incorrect answers immediately
- ✅ **Progress Tracking** - View personal quiz history and scores
- ✅ **Best Scores** - Track your best performance

### 🎨 Design Features
- ✅ **Animated Starry Background** - Beautiful twinkling stars and shooting stars
- ✅ **Glass Morphism Cards** - Modern, professional UI design
- ✅ **Marquee Scrolling Text** - Engaging welcome messages
- ✅ **Fully Responsive** - Works on desktop, tablet, and mobile
